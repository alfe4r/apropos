# Apropos – Estações de IA

Site institucional estático gerado com Next.js App Router, Tailwind CSS e conteúdo em MDX.

## Requisitos

- Node.js 18+
- npm

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy na Vercel

1. Faça o push do repositório para o GitHub.
2. Conecte o repositório na Vercel.
3. Use as configurações padrão de Next.js.
4. Clique em **Deploy**.

O projeto está pronto para SSG e publicação direta.
